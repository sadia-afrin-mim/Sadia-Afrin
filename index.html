Research Works

Assisting practitioners in effectively utilizing machine learning fairness tools (PhD Research)

In my research, I am studying how to make machine learning fairness tools more functioning to practitioners. I developed a taxonomy of existing fairness tools. This study critically assesses and classifies the current landscape of machine learning fairness tools, culminating in the creation of a practical taxonomy. The taxonomy aims to assist both practitioners and researchers in comprehending the existing tools, enabling them to identify any existing gaps within the field. Currently, I am working on a user study of the practitioner's perspective on these tools and aiming to use these perspectives to develop more sophisticated machine learning fairness tools.

Developing Deep Neural Network Causal Testing Tool (Research Project)

In this work, I am focusing on developing casual testing tools for deep neural networks. This work generates fuzzy inputs on a failed test. Filtering these inputs user can find most similar 3 passing and failing inputs that help developers to understand root cause of a test failure. Currently I am working on including fairness testing to this work.
